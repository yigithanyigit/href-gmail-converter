# Mailto-Gmail-Converter Chrome Extension
Mailto-Gmail-Converter is a Chrome extension designed to convert mailto: URIs into Gmail URLs. This allows users to send emails via Gmail's web interface instead of using the default email client.

# Main Features
- Converts mailto: URIs into Gmail URLs
- Overrides the default mailto: URI behavior
- Enables sending emails via Gmail's web interface
# How to Use
- Install the Mailto-Gmail-Converter extension.
- Click on any mailto: link on a webpage.
- The extension will automatically convert the mailto: URI into a Gmail URL and open a new Gmail compose window.
# Development
This extension is comprised of the following files:

- manifest.json: The manifest file for the extension.
- injector.js: The content script that is injected into web pages to convert mailto: links.
- popup.html: The HTML file for the popup interface (if any).
- background.js: The background script that runs in the background (if any).
